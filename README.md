**TerraFirmaCraft Remake**（译为群峦传说：重制）是一款偏向原生整合的群峦整合包。  
本整合包的功能尽量以数据包，KubeJS 脚本来实现一些有趣的特性。

---
**TerraFirmaCraft Remake** is a TFC ModPack that leans towards vanilla integration.    
The functionalities of this ModPack are implemented as much as possible using data packs and KubeJS scripts to introduce some interesting features.
